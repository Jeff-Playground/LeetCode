This is a data structure built upon the suffix array. It is a special suffix array, where it not only stores the sorted
indices, but also stores the longest common prefix at each location(the first value is defaulted to 0 and doesn't mean
anything).

This data structure is useful for below cases:
Find all distinct substrings:
    https://www.youtube.com/watch?v=m2lZRmMjebw&list=PLDV1Zeh2NRsCQ_Educ7GCNs3mvzpXhHW5&index=3
Find the longest repeated substring within one string:
    https://www.youtube.com/watch?v=OptoHwC3D-Y&list=PLDV1Zeh2NRsCQ_Educ7GCNs3mvzpXhHW5&index=6
Find the longest common substrings among multiple strings(it could be appearing in k strings, where 2<=k<=n):
    https://www.youtube.com/watch?v=Ic80xQFWevc&list=PLDV1Zeh2NRsCQ_Educ7GCNs3mvzpXhHW5&index=4
